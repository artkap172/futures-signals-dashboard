# 📊 Дашборд торговых сигналов по фьючерсам MOEX (v3.0)

Автоматизированная система генерации торговых сигналов для ликвидных фьючерсов Московской биржи с продвинутой аналитикой и риск-менеджментом.

🚀 **Демо:** [https://artkap172.github.io/futures-signals-dashboard/](https://artkap172.github.io/futures-signals-dashboard/)

## 📊 Отслеживаемые инструменты
* **BR** - Нефть Brent
* **NG** - Природный газ
* **GD** - Золото
* **SV** - Серебро
* **Si** - USD/RUB
* **MX** - Индекс MOEX

## ✨ Ключевые особенности v3.0
* **Динамический риск-менеджмент:** Стоп-лосс и Тейк-профит рассчитываются на основе волатильности (ATR 14).
* **Адаптация к режиму рынка:** Определение фазы тренда (Trend/Flat) через EMA 100/200 и фильтрация сигналов.
* **Вероятностный скоринг:** Оценка уверенности сигнала на основе комбинации RSI, MACD и ценовых паттернов.
* **Статистика:** Отображение исторического Win Rate для каждого типа сигнала.
* **Обновление каждые 15 минут:** Полная автоматизация через GitHub Actions.

## 🛠 Технический стек
* **Python 3.10:** Скрипт генерации сигналов (`generate_signals.py`).
* **HTML5/CSS3/JS:** Адаптивный фронтенд с динамическим обновлением.
* **GitHub Actions:** CI/CD для регулярного сбора данных из MOEX ISS API.

## 📖 Как это работает
1. Скрипт подключается к API Московской Биржи и загружает исторические свечи (OHLC).
2. Рассчитываются индикаторы: RSI, MACD, EMA (20, 50, 100, 200).
3. Определяется режим рынка и волатильность инструмента.
4. Генерируется сигнал с конкретными точками входа и выхода.
5. Результаты сохраняются в `signals.json` и отображаются на дашборде.
