# 📈 Дашборд торговых сигналов по фьючерсам MOEX

## 🎯 О проекте

Автоматизированная система генерации торговых сигналов для ликвидных фьючерсов Московской биржи.

**🌐 Демо:** https://artkap172.github.io/futures-signals-dashboard/

## 📊 Отслеживаемые инструменты

- **BR** - Нефть Brent
- **NG** - Природный газ
- **GD** - Золото
- **SV** - Серебро  
- **Si** - USD/RUB
- **MX** - Индекс MOEX

## ✨ Возможности

✅ Технический анализ с индикатором RSI  
✅ Автоматическая генерация сигналов LONG/SHORT  
✅ Расчет точек входа, Stop-Loss и Take-Profit  
✅ Оценка соотношения риск/прибыль  
✅ Уровень уверенности для каждого сигнала  
✅ Обновление каждые 15 минут через GitHub Actions  
✅ Responsive веб-интерфейс

## 🛠️ Структура проекта

```
futures-signals-dashboard/
├── index.html              # Главная страница дашборда
├── signals.json            # JSON с сигналами (автообновляется)
├── generate_signals.py     # Скрипт генерации сигналов
├── requirements.txt        # Python зависимости
└── .github/workflows/
    └── update-signals.yml  # GitHub Actions для автообновления
```

## 🚀 Как работает

1. **GitHub Actions** запускается каждые 15 минут
2. **generate_signals.py** подключается к API Московской биржи
3. Загружаются актуальные данные по фьючерсам
4. Рассчитывается индикатор RSI (14)
5. Генерируются сигналы на основе технического анализа
6. Обновляется **signals.json**
7. **index.html** автоматически отображает новые данные

## 📝 Формат сигнала

Каждый сигнал содержит:

- **Тип:** LONG или SHORT
- **Вход:** Диапазон цен для входа
- **Stop-Loss:** Уровень защитного стопа
- **Take-Profit 1:** Первая цель
- **Take-Profit 2:** Вторая цель
- **Риск/Прибыль:** Соотношение R:R
- **Уверенность:** Процент уверенности в сигнале
- **Комментарий:** Дополнительная информация

## 🔧 Локальная установка

### Требования
- Python 3.10+
- pip

### Шаги установки

```bash
# Клонировать репозиторий
git clone https://github.com/artkap172/futures-signals-dashboard.git
cd futures-signals-dashboard

# Установить зависимости
pip install -r requirements.txt

# Запустить генерацию сигналов
python generate_signals.py

# Открыть index.html в браузере
```

## ⚙️ Настройка GitHub Actions

Для автоматического обновления сигналов GitHub Actions уже настроен:

- **Расписание:** Каждые 15 минут (`*/15 * * * *`)
- **Ручной запуск:** Через вкладку Actions → Run workflow

### Права доступа

Убедитесь, что в Settings → Actions → General включено:
- ✅ Read and write permissions
